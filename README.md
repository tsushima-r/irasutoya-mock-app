# セリフ選択UIモック 🎮🗨️

RPGやビジュアルノベル風ゲームによく登場する、  
「セリフ選択」UIを再現したモックです。

## 🔍 概要

- 選択肢によって背景やテキストが変化する分岐形式
- JavaScriptのみで実装されたシンプルな構造
- 戻るボタンによる履歴管理（`history.pushState`）
- 背景画像とセリフにより、ちょっとした世界観も演出
